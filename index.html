🚀 Sobre o Aplicativo

  • Integração Direta (WMS + SAGA): O aplicativo se conecta à API interna do SAGA, trazendo os eventos em tempo real para a mesma tela do WMS, eliminando a necessidade de buscar em outras abas.
  • Auto-Preenchimento Manual: Na aba de inserção manual, basta colar o código da TOTE pura. O app faz a busca invisível e preenche a Localização, o Aging e o ETD nativamente para você.
  • Cópia Inteligente para o Sheets: Botão exclusivo que formata os dados para o Google Sheets. Ao colar na planilha, os dados caem perfeitamente divididos nas colunas de forma automática.
  • Motor em Nuvem: O aplicativo recebe atualizações de melhorias automaticamente de forma silenciosa via GitHub.

🛠️ Passo a Passo para Instalação

Passo 1: Instalar e Configurar o "Motor"
O navegador precisa do Tampermonkey para rodar o painel no WMS. Após a instalação, é obrigatório configurar o modo desenvolvedor:

1. Instale a extensão no botão abaixo.
2. Abra uma nova aba e digite chrome://extensions
3. No canto superior direito, ative a chave "Modo do desenvolvedor". Se não fizer isso, o Chrome bloqueará o App!
1. Instalar Tampermonkey no Chrome
Passo 2: Ativar o Aplicativo Oficial
Com a extensão instalada e o Modo Desenvolvedor ativo, clique no botão abaixo. Uma tela do Tampermonkey vai se abrir, basta clicar no botão verde "Instalar" (ou "Atualizar").
2. Ativar Aplicativo de Rastreio
